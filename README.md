# Trader-Sentiment-Analysis
Data analysis project exploring how Bitcoin market sentiment (Fear &amp; Greed Index) influences trader behavior and performance using Python and Google Colab.

## How to Set Up and Run the Project (Google Colab)

This project analyzes trader performance based on Bitcoin market sentiment using Python and Google Colab.

---

## Software Requirements

You only need:

* A Google account
* A web browser (Chrome recommended)
* Internet connection

No local installation is required.

---

## Files Required

Make sure you have the following files downloaded from the repository:

* `Trader_Sentiment_Analysis.ipynb`
* `fear_greed_index.csv`
* `historical_data.csv`

All three files are required to run the project.

---

## Steps to Run the Project

### Step 1: Open Google Colab

Go to:
https://colab.research.google.com

---

### Step 2: Upload the Notebook

1. Click **File → Upload Notebook**
2. Select `Trader_Sentiment_Analysis.ipynb`

---

### Step 3: Upload Dataset Files

1. On the left sidebar, click the **Files icon**
2. Click **Upload**
3. Upload:

   * `fear_greed_index.csv`
   * `historical_data.csv`

Make sure both files appear in the file panel.

---

### Step 4: Run the Notebook

1. Click **Runtime → Run All**
2. The notebook will automatically:

   * Load the datasets
   * Perform data cleaning
   * Merge data
   * Generate metrics
   * Display analysis charts

---

## Output Generated

After running all cells, the notebook will produce:

* Daily Profit & Loss analysis
* Trade activity comparison by sentiment
* Win rate analysis by sentiment
* Visual charts showing trader behavior patterns

All outputs are displayed directly within the notebook.

---

## Troubleshooting

If errors occur:

* Ensure dataset files are uploaded before running
* Check that file names match exactly
* Restart runtime and run all cells again if needed

---

This project is designed to run entirely in Google Colab without any additional software installation.
